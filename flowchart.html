📋 صفحات النظام

  • 📂 صفحة التصنيفات: عرض جميع التصنيفات المتاحة (إيميلات، مراسلات، شهادات، إلخ).
  • 📜 صفحة التصنيف: عرض النماذج داخل التصنيف مع إمكانية الإضافة والتعديل والحذف.
  • 📄 صفحة النموذج: عرض النموذج المختار مع المتغيرات المطلوبة.
  • 📥 إدخال المتغيرات: إدخال القيم يدويًا أو تحميلها من ملف Excel.
  • 💾 الحفظ والإرسال: اختيار بين عملية فردية أو مجمعة واستبدال البيانات تلقائيًا.

📌 خطوات العملية

  1. الدخول إلى صفحة التصنيفات.
  2. إنشاء أو اختيار تصنيف معين.
  3. عرض النماذج داخل التصنيف.
  4. اختيار نموذج معين.
  5. إدخال القيم الخاصة بالمتغيرات يدويًا أو عبر ملف Excel.
  6. اختيار بين عملية فردية أو مجمعة.
  7. في العملية المجمعة، يتم تحليل بيانات Excel واستبدالها تلقائيًا.
  8. حفظ النماذج النهائية أو إرسالها.

🔧 إضافات إدارية

  • لوحة إدارة (Admin Panel) لإدارة المستخدمين.
  • صفحة الملف الشخصي لكل مستخدم.
  • خيار "هل نسيت كلمة السر؟".
